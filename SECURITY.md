# Security & Privacy Policy

Top Monitoring is designed to be fully local and transparent.

## Data Collection
- ❌ No personal data collected
- ❌ No telemetry
- ❌ No analytics
- ❌ No logging to external services

## Network Usage
- The application does NOT send any data over the internet.
- Network metrics are read locally from OS counters.

## Permissions
- No administrator privileges required
- No background services
- No startup modification by default

## Third-party Libraries
- LibreHardwareMonitor (hardware sensors)
- .NET / WPF runtime

## Reporting Security Issues
If you find a security issue:
- Open a GitHub Issue
- Or contact the maintainer directly

Thank you for helping keep the project safe.
